# Dating app
## _Generic dating app_

Dating social network developed in Asp.net core 5 and Angular 11

## Features

- Register/Login feature
- Real time chat
- Import/Export images
- Create/Watch user's profile
- More... 

## Requirements

- .Net Core 5.0.3
- Npm 6.14.10
- Node 14.15.4
